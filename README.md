# 邀请码小助手

一个简洁的邀请码管理工具，支持图片识别，一键复制跳转。

## 使用方法

### 1. 上传截图识别

点击「上传截图」按钮，选择包含邀请码的图片，系统会自动识别出邀请码。

> **提示**：部分图片可能识别失败，这时需要手动输入邀请码。

### 2. 手动添加

直接在输入框输入9位邀请码（支持以7、8、9开头），点击添加即可。

### 3. 一键使用

- 点击邀请码可直接复制
- 点击「跳转」按钮可直接打开活动页面
- 使用后点击「已使用」标记，该邀请码会自动移除

## 注意事项

- 邀请码必须为 **7、8、9开头** 的 **9位数字**
- 每个IP每天最多添加 20 个邀请码
- 邀请码每天自动清空，请及时使用
- 已使用的邀请码会自动删除

## 常见问题

**Q: 识别不准确怎么办？**
A: 部分图片识别可能失败，请手动输入邀请码。

**Q: 提示"操作过于频繁"？**
A: 请等待一分钟后再试，或明天继续使用。

**Q: 提示"今日操作次数已达上限"？**
A: 每个IP每天有操作次数限制，请第二天再来。
