# PDD 邀请码识别工具

## 项目概述

这是一个基于 Node.js 的拼多多邀请码识别工具，支持上传截图自动识别邀请码并管理。

### 技术栈

- **后端**: Node.js + Express
- **OCR引擎**: Tesseract.js v5
- **图像处理**: Sharp
- **前端**: 原生 HTML/CSS/JavaScript

### 核心功能

1. **图片上传识别**: 上传截图自动识别邀请码
2. **自动保存**: 识别成功后自动保存邀请码
3. **邀请码管理**: 标记已使用、查看历史记录
4. **一键跳转**: 直接跳转到拼多多APP

## 构建和运行

### 安装依赖

```bash
npm install
```

### 启动服务

```bash
npm start
```

服务默认运行在端口 `8080`，可通过环境变量 `PORT` 修改。

### 环境变量

| 变量名 | 默认值 | 说明 |
|--------|--------|------|
| PORT | 8080 | 服务端口 |
| DATA_DIR | . | 数据存储目录 |

## API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/` | 首页 |
| GET | `/api?action=get` | 获取邀请码列表 |
| POST | `/api/ocr` | 上传图片识别邀请码 |
| POST | `/api/mark_used` | 标记邀请码已使用 |

## OCR 识别参数

图像处理流程：
1. 裁剪底部 35% 区域（邀请码位置）
2. 缩放至最大宽度 800px
3. 灰度化处理
4. 二值化（阈值 200）

识别规则：
- 只识别以 `7` 开头的 9 位数字
- 非 7 开头的识别结果不保存

## 数据存储

邀请码数据存储在 `codes.json` 文件中，每天自动清理过期数据。

### 数据结构

```json
{
  "id": 1,
  "number": "774270390",
  "timestamp": 1772500835678,
  "date": "2026-03-03",
  "used": false
}
```

## 开发注意事项

- 邀请码必须为 7 开头的 9 位数字
- 数据每天自动清空，只保留当天数据
- Tesseract Worker 在服务启动时预加载
